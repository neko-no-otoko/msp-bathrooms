# Minneapolis Bathroom Tracker

A community-driven web application to track finding publicly accessible bathrooms in Minneapolis.

## Features

- **Map View**: Interactive map showing bathroom locations.
- **List View**: Browse bathrooms in a list format.
- **Suggest a Bathroom**: Form for users to submit new locations.
- **Vote/Upvote**: (Planned) Community validation of bathroom quality and accessibility.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Data**: JSON (Local storage/simulation)
- **Map**: Leaflet.js / OpenStreetMap

## Setup

1. Clone the repository.
2. Open `msp-bathrooms/index.html` in your browser.
